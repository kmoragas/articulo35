Fotoreportaje

SecureDrop es una herramienta que nos va a permitir compartir consejos, información y archivos cuya importancia es muy sensible y exige un mayor grado de anonimato.
Para proteger su anonimato se utilizará la red Tor este no registra su dirección IP o información acerca de su navegador, ordenador o sistema operativo.
Red Tor:  
https://torproject.org

¿Cómo utilizar este anonimato de Articulo35?

*  Coloca en tu navegador y sigue las instrucciones para descargar Tor, escogiendo la versión del sistema operativo correspondiente en el siguiente link: 
 
*  Ejecuta la aplicación de Tor, una vez que carga el navegador Tor, copie y pegue esta url en la barra de direcciones de Tor. URL: 

* Una vez cargada la página se le mostrarán las instrucciones para poder compartir su información con nosotros.

Consejos para usar este anonimato

* Utilice preferiblemente una red WI-FI privada.

* Si utiliza una red publica tener cuidado de que nuestra pantalla no sea visible por las cámaras de seguridad.

* Otra alternativa es usar el sistema operativo Tails que pueden ser arrancado desde una llave USB.
